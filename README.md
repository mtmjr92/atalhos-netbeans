# Atalhos Uteis Netbeans

### EXECUCAO

- F6 - Executar
- SHIFT + F11 - Limpar e Construir 
- F11 - Construir

### DEPURACAO

- CTRL + F5 - Depurar
- CTRL + F9 - Analisar Expressao 

#### CODIGO FONTE
- CTRL + SHIFT + (-) - Recolher todas as dobras
- ALT + SHIFT + F - Formatar
- CTRL + SHIFT + I - Remover importes
